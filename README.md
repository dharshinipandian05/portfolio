- `src/components/Hero.tsx` — Dynamic greeting and custom entry animations.

## ⚙️ Local Development Settings

### Prerequisites
- Node.js (version 18 or above recommended)
- npm or yarn

### Installation
Clone this repository and download current dependencies:
```bash
npm install
```

### Run Locally (Development server)
```bash
npm run dev
```

### Build for Production
Compiles final static assets into the `/dist` output folder:
```bash
npm run build
```

### Code Quality Checklist (Linting)
Run TypeScript transpilation checks of the portfolio codebase:
```bash
npm run lint
```
